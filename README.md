# Simple Landing Page

A modern, responsive landing page built while learning Claude Code.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile
- **Smooth Animations**: Fade-in effects and hover animations
- **Interactive Elements**: Smooth scrolling navigation and button effects
- **Modern Styling**: Gradient hero section with card-based layout

## Files

- [index.html](index.html) - Main HTML structure
- [styles.css](styles.css) - All styling and animations
- [script.js](script.js) - Interactive JavaScript functionality

## How to View

Simply open `index.html` in your web browser!

## What's Included

- Fixed navigation bar with smooth scrolling
- Hero section with call-to-action button
- Features section with three cards
- Contact section
- Footer

Built with ❤️ using Claude Code
